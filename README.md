# Adrenalin

[See demo](https://adrenalin-three.now.sh/)

## Technologies Used

- ReactJS
- Redux
- Reactstrap
- Chai
- Enzyme

## Setting Up

First, you need to install all dependencies by running:

```
npm install
```

After successful installation, start the app by running:

```
npm run start
```

For unit testing

```
npm run test
```

To get the production built version,

```
npm run build
```
